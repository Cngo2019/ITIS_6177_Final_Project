About the Endpoint

This endpoint removes the background from an uploaded image. The returned result is a PNG image with the background removed.

Full URL Path

POST http://146.190.147.242:3000/api/remove-background

Headers

{
  "Content-Type": "multipart/form-data"
}

Request Body

An image file uploaded as multipart/form-data. The field name should be photo.

Example API Call (Postman)

Follow these steps to test the /remove-background endpoint using Postman:

  1. Open Postman and create a new POST request.
  2. Set the URL to: http://146.190.147.242:3000/api/remove-background (adjust if your server is hosted elsewhere).
  3. Under the Headers tab, verify that you have:
    • Key: Content-Type
    • Value: multipart/form-data
      • Postman should already set the header and boundary for you.
  4. Switch to the Body tab and select form-data.
    • Add a new key called photo.
    • Set the type to File and upload your image file.
  5. Click Send to submit your request. The response will be an image (PNG format) with the background removed.

Error Codes

If something goes wrong, the API will respond with appropriate error codes:

  • 400 Bad Request — No file was uploaded. Ensure that the request contains a file named photo in the form-data.
  • 500 Internal Server Error — An unexpected error occurred on the server.
    Occurs when:
    • An invalid file format is uploaded (e.g., a non-image file like a PDF).

    • The image dimensions are too small or too large for processing.
      Azure requires images to be between 50×50 and 16,000×16,000 pixels

    • An error occurred while communicating with the Azure Computer Vision service.
